# Workstation

This repository documents the development environment I intentionally maintain on my Mac.

Instead of treating my computer like a collection of random tools, I want to understand exactly what I have installed, why it's there, and how it supports the projects I build.

This isn't a backup of my computer or a list of everything I've ever installed.

It's a living reference for the development tools, configurations, and workflows I intentionally maintain.

As I build new types of software, this repository will grow alongside my development environment.
